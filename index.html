Welcome to the wonderful StreetHub API

StreetHub is a click-and-collect network of local boutiques with the best of design and fashion. We allow people to discover things they love, near them and reserve any item they would like to purchase.

Through this API you have access to:

  • About 100 boutiques that are part of the StreetHub network. (Images, location, address, general information about the boutiques etc.)

  • Around 2000 products that these boutiques have published through our inventory management system. All of these products are in-stock and ready to be collected. (Images, descriptions, price, location, stock status etc.)

You'll be able to retrieve lists of boutiques and products and search by keyword and location for specific datasets. This page will provide you with information about the two main resources of the API - boutiques & products

If you need any support let us know on tech@streethub.com

Base URL

https://www.streethub.com/

Products

GET api/public/products

Returns the products that correspond to your query. if no query parameters are set, returns recommended products.

Parameters
parameter description
size set size of request. maximum of 48.

Example Values: 2 6
page  the next page for pagination. The second page is reffered to by 1.

Example Values: 1 2 3
search search query to match against.

Example Values:
category Filter by category. You can select multiple categories by using category[]=jewellery&category[]=kitchen.

Valid Values: living, kitchen, bathroom, bedroom, outdoor, study, gifts, jewellery, mens, womens, other, sale
area Filter by area. You can select multiple areas by using area[]=hampstead&area[]=twickenham.

Valid Values: chiswick, crouch-end, twickenham, barnes, portobello-road, columbia-road, kings-road, brick-lane, dulwich, hammersmith, stratford, muswell-hill, camden, mayfair, bermondsey, clerkenwell, greenwich, hampstead, hackney, marylebone, chelsea-kensington, soho, notting-hill, bloomsbury, islington, stoke-newington, southwark, shoreditch
lat, lon, radius Returns all products within the radius of that location. Radius in miles. If not set, radius defaults to 100.

Example Values: lat=50.0102, lon=-0.01434, radius=5
sortBy Sort results. Sorting by distance only makes sense if you provide lat and lon as well. If not set, defaults to recommended.

Valid Values: price_asc, price_desc, distance_asc, recommended.
Special cases:

These special cases can't be used with the other parameters as they are used for very specific use cases

parameter description
boutique_slug returns all products of this boutique. boutique_slugs are unique across the database.

Example Values: abigail-ahern-in-n11qp
slug, boutique_slug returns the single product of this boutique with this slug.

Example Values: slug=porcelain-tea-light-holder-neon-pink, boutique_slug=abigail-ahern-in-n11qp

Common Errors

> Arghs! Origin is not allowed by Access-Control-Allow-Origin.

For the public API we enabled jsonp, use it like so:

Example jQuery:
$.ajax({
    url: 'https://www.streethub.com/api/public/products',
    type:'GET',
    dataType:'JSONP',
    success: function(data){
    }, 
    error: function(data) {
    }
});

This is enabled for both /boutiques and /products.
